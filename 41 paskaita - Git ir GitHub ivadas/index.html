Terminalo Komandos

  • ls / dir - parodo aplankalus ir failus dabartineje direktorijoje
  • cd - changeDirectory - keicia direktorija
  • TAB - skirtas uzpildyti terminalo komanda arba pateikti pasiulymus
  • CTRL C - atsaukia vykdoma veiksma / nutraukia komanda
  • vidurinis/desinys peles mygtukas - iklijuoja teksta i komandine eilute
  • ARROW UP/DOWN - paraso pries tai naudots komanda/vaiksciojimas per terminalo komandu istorija

Git Komanda

  • git init - inicializuoja git repozitorija
  • git status - parodo git repozitorijos statusa
  • git add - prideti failus prie paruostu commit'inimui failu
  • git commit -m - "zinute" sukuriama kodo versija su zinute
  • git restore - panaikina dar nekomitintus pakeitimus
  • git restore --stage ./*/file - isima nurodyta faila is stage zonos
  • git push - nusiuncia naujus commit'us i serveri
  • git log - rodo git commit'u istorija
  • git checkout commID - nukrliauja i nurodyta kodo versija
